## üîó API Endpoints

### **Authentication**

* `POST /api/v1/user/signup` ‚Üí Register new user
* `POST /api/v1/user/login` ‚Üí Login user
* `GET /api/v1/user/getUser` ‚Üí Get logged-in user details

### **Products**

* `GET /api/v1/products` ‚Üí Get products (with pagination)
* `GET /api/v1/products/:id` ‚Üí Get single product
* `POST /api/v1/products` ‚Üí Create new product
* `PATCH /api/v1/products/:id` ‚Üí Update product
* `DELETE /api/v1/products/:id` ‚Üí Delete product

## üöÄ Getting Started

### **Prerequisites**

* Node.js (v14+)
* npm or yarn
* MongoDB (local or cloud instance)

### **Setup**

```bash
# Clone repo
git clone https://github.com/yourusername/Catalogo.git
cd Catalogo
```

#### **Backend**

```bash
cd backend
npm install
```

Create `.env` file:

```env
PORT=8000
MONGO_URI=your_mongodb_uri
FRONTEND_URL=http://localhost:5173
JWT_SECRET=your_jwt_secret
```

Start server:

```bash
npm run dev
```

#### **Frontend**

```bash
cd frontend
npm install
```

Create `.env` file:

```env
VITE_BACKEND_URL=http://localhost:8000/api/v1
```

Run frontend:

```bash
npm run dev
```
