## üîó API Endpoints

### **Authentication**

* `POST /api/v1/user/signup` ‚Üí Register new user
* `POST /api/v1/user/login` ‚Üí Login user
* `GET /api/v1/user/getUser` ‚Üí Get logged-in user info (protected)

### **Products**

* `GET /api/v1/products` ‚Üí Get all products
* `GET /api/v1/products/:id` ‚Üí Get product by ID
* `POST /api/v1/products` ‚Üí Create new product (admin or owner)
* `PUT /api/v1/products/:id` ‚Üí Update product (admin or owner)
* `DELETE /api/v1/products/:id` ‚Üí Delete product (admin or owner)

## üöÄ Getting Started

### **Prerequisites**

* Node.js (v14+)
* npm or yarn
* MongoDB (local or cloud instance)

### **Setup**

```bash
# Clone repo
git clone https://github.com/your-username/catalogo.git
cd catalogo
```

#### **Backend**

```bash
cd backend
npm install
```

Create `.env` file:

```env
PORT=5000
MONGO_URI=your_database_uri
FRONTEND_URL=http://localhost:5173
JWT_SECRET=your_jwt_secret
```

Start backend:

```bash
npm run dev
```

#### **Frontend**

```bash
cd frontend
npm install
```

Create `.env` file:

```env
VITE_BACKEND_URL=http://localhost:5000/api/v1
```

Start frontend:

```bash
npm run dev
```
